Software as a Service (SaaS)

What is Software as a service?(Salesforce): Software as a service allows companies to adopt cutting edge software applications to their businesses while also not having to deal with the disadvantages that comes with it such as maintaining complex hardware and software locally.

How does this work? SaaS works by being accessed over the internet. The software will be stored on the providers servers while the clients may access the applications through a web browser.

Platform as a Sevice (PaaS)

What is Platform as a service? Platform as a service is typically an operating system or programming language platform which allows users to create programming projects online while not having to store any of the code locally, instead saving it online in the cloud.

What are the advantages to PaaS? Paas holds many advantages; the main ones being that it provides access to a range of coding languages on its platform to people who may not have access on their own hardware. Another advantage is that it allows users to access their coding projects on any device as long as they have internet access.
